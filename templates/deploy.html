1Prepare Your Repository

First, make sure your code is on GitHub:

# Initialize git repository (if not already done) git init git add . git commit -m "Initial commit - StrokeCare AI" # Create a new repository on GitHub, then: git remote add origin https://github.com/YOUR_USERNAME/stroke-prediction-app.git git branch -M main git push -u origin main
üìù Note: Make sure to include these files in your repository:
  • streamlit_app.py
  • stroke_binary_model.pkl
  • stroke_probability_model.pkl
  • requirements.txt

2Deploy on Streamlit Cloud

  1. Go to share.streamlit.io
  2. Sign in with your GitHub account
  3. Click "New app"
  4. Select your repository: YOUR_USERNAME/stroke-prediction-app
  5. Set the main file path: streamlit_app.py
  6. Click "Deploy"
‚úÖ That's it! Your app will be live at:
https://YOUR_APP_NAME.streamlit.app

3Verify Model Files

Make sure your model files are in the repository root:

4Test Your Deployment

Once deployed, test your app:

  1. Open your Streamlit app URL
  2. Fill in the patient information form
  3. Click "Assess Stroke Risk"
  4. Verify results are displayed correctly

1Install Heroku CLI

Download and install from: devcenter.heroku.com

2Create Procfile

Create a file named Procfile (no extension) in your project root:

web: gunicorn app:app

3Deploy

heroku login heroku create your-app-name git push heroku main

1Install Railway CLI

npm i -g @railway/cli

2Deploy

railway login railway init railway up

1Create Dockerfile

Create a file named Dockerfile:

FROM python:3.11-slim WORKDIR /app COPY requirements.txt . RUN pip install -r requirements.txt COPY . . EXPOSE 8501 CMD ["streamlit", "run", "streamlit_app.py", "--server.port=8501", "--server.address=0.0.0.0"]

2Build and Run

docker build -t stroke-app . docker run -p 8501:8501 stroke-app
